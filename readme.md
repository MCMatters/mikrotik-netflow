## Overview

This tool will help you track all your traffic through Netflow protocol. This is alpha version.

## Setup

* Connect to your Mikrotik and setup Traffic Flow. Use `8787` or setup your own custom port to start tracking traffic
* Run `docker-compose up -d`. If you wish, you can expose `mongo` port `27017`
